# Groove Vault

A full-stack practice tracker for drummers to log rudiments, track BPM progress, 
and stay consistent with a built-in metronome.

## Tech Stack
- React, TypeScript, Tailwind CSS
- Node.js, Express
- PostgreSQL

## Running the Project

**Backend**
```
cd backend
npm install
npm run dev
```

**Frontend**
```
cd frontend
npm install
npm run dev
```

> Requires a `.env` file with `DATABASE_URL` in the backend folder.
